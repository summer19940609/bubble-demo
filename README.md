# AI消息滚动Demo

使用 antd-design-x 2 的 bubble.list 和 react-infinite-scroll-component 实现的AI消息滚动演示。

## 功能特性

- 使用 `@ant-design/x` 的 `Bubble.List` 和 `Bubble` 组件展示消息气泡
- 使用 `react-infinite-scroll-component` 实现无限滚动加载
- 模拟AI消息的实时流式输出
- 支持用户和AI助手两种角色的消息展示
- 自动滚动到最新消息

## 安装依赖

```bash
pnpm install
```

## 运行项目

```bash
pnpm dev
```

## 技术栈

- React 18
- TypeScript
- Vite
- Ant Design 5
- @ant-design/x 2
- react-infinite-scroll-component
